🚀 独立终端输入框演示

单独的终端输入框，支持自动补全和AI补全功能

📋 功能特性

  • 独立输入框：独立的终端输入界面，不干扰终端显示
  • 自动补全：输入时显示智能建议列表
  • AI智能建议：集成AI提供个性化命令建议
  • 本地命令库：包含100+常用Linux命令
  • 键盘导航：Tab补全，方向键选择，Esc取消
  • 历史记录：自动保存和浏览命令历史
  • 实时反馈：命令执行结果实时显示

⌨️ 快捷键操作

  • Tab - 自动补全选中的建议
  • ↑↓ - 导航建议列表/历史命令
  • Esc - 隐藏建议列表
  • Ctrl+Space - 显示/隐藏补全建议
  • Ctrl+↑↓ - 浏览命令历史
  • F4 - 切换AI补全开关
  • Enter - 执行命令
