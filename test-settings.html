SettingsModal 组件测试

🎯 测试目标

验证AI聊天和AI补全表单的布局样式修复，以及新增的自定义设置功能。

✨ 新增功能

  • 🔧 自定义 Base URL 设置
  • 🔑 API Key 显示/隐藏切换
  • 🤖 自定义模型名称输入
  • 📋 预设模型快速选择
  • 🔄 提供商切换时自动填充默认配置
  • 📱 响应式表单布局优化

🧪 测试步骤

  1. 点击"打开设置面板"按钮
  2. 切换到"AI聊天"标签页
  3. 测试不同提供商的切换（OpenAI、Anthropic、Local、自定义）
  4. 测试API Key的显示/隐藏功能
  5. 测试自定义Base URL输入
  6. 测试模型名称的自定义和预设选择
  7. 切换到"AI补全"标签页重复上述测试
  8. 测试设置保存和重置功能
