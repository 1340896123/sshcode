# SSH Code 核心概念定义

本文档定义了 SSH Code 应用中的核心概念，为开发团队提供统一的术语和架构理解。

## 核心概念定义

### 1. 链接 (Connection)

**定义**：链接代表一个独立的SSH连接实例，包含与远程服务器的完整通信会话。

**核心属性**：
- **唯一标识符**：UUID v4 格式的全局唯一ID
- **服务器信息**：主机地址、端口(默认22)、用户名
- **认证方式**：支持密码认证和密钥认证两种方式
- **连接状态**：`disconnected` | `connecting` | `connected` | `failed` | `reconnecting`
- **健康监控**：延迟监测、传输字节数统计、内存使用监控
- **自动重连**：可配置的重连策略和最大重试次数

**技术实现**：
```typescript
interface Connection {
  id: string;
  name: string;
  host: string;
  port: number;
  username: string;
  authType: 'password' | 'key';
  status: ConnectionStatus;
  latency?: number;
  bytesTransferred: { sent: number; received: number };
  memoryUsage: number;
  healthStatus: 'healthy' | 'unhealthy' | 'failed';
  // ... 其他属性
}
```

**特点**：
- 每个链接都是完全独立的，支持并发操作
- 允许到同一服务器的多个并发链接
- 具备故障恢复和健康检查能力
- 资源使用监控和限制管理

### 2. 会话 (Session)

**定义**：会话是链接在特定标签页中的具体实例，包含该链接的所有状态和上下文信息。

**核心组件**：

#### 2.1 终端会话 (Terminal Session)
- **命令历史**：完整的命令执行记录
- **输出缓冲区**：终端输出内容缓存(最多1000行)
- **光标位置**：当前光标在终端中的坐标
- **工作目录**：会话当前所在的文件系统路径
- **Shell环境**：环境变量和Shell配置

#### 2.2 文件管理会话 (File Manager Session)
- **当前目录**：文件管理器显示的目录路径
- **导航历史**：目录浏览的历史记录栈
- **选中文件**：当前选中的文件列表
- **书签管理**：用户保存的常用目录快捷方式
- **传输状态**：文件上传/下载的进度和状态

#### 2.3 AI助手会话 (AI Assistant Session)
- **对话历史**：用户与AI的完整聊天记录
- **工具调用**：AI执行的命令和工具使用记录
- **上下文信息**：工作目录、环境变量、系统信息
- **建议缓存**：基于历史数据的智能建议

**隔离特性**：
- 每个会话拥有独立的工作目录和环境变量
- 会话间完全隔离，互不影响
- 支持会话状态的持久化和恢复
- 资源使用独立计费和限制

### 3. 标签栏 (Tab Bar)

**定义**：标签栏是用户界面的导航组件，用于管理和切换多个会话。

**功能特性**：
- **会话显示**：显示所有活跃会话的缩略信息
- **状态指示**：可视化连接状态和活动状态
- **拖拽排序**：支持鼠标拖拽重排序标签
- **上下文菜单**：右键操作菜单(重命名、关闭、复制等)
- **快捷键支持**：键盘快捷键导航(Ctrl+Tab, Ctrl+1~9)

**交互设计**：
- 点击标签立即切换到对应会话
- 关闭标签时显示确认对话框(如有活动操作)
- 标签名称可自定义编辑
- 支持标签分组和颜色标识

**视觉设计**：
- 活动标签高亮显示
- 连接状态用颜色图标表示
- 标签宽度自适应内容
- 支持滚动显示多标签

### 4. 文件目录 (File Directory)

**定义**：文件目录是文件管理器组件，提供远程服务器文件系统的可视化和操作界面。

**状态管理**：
- **当前路径**：完整的工作目录路径
- **目录历史**：支持前进/后退的导航历史
- **文件列表**：当前目录的文件和文件夹列表
- **选中状态**：单选或多选的文件项目
- **视图模式**：列表视图和网格视图切换

**操作功能**：
- **文件浏览**：目录树导航和文件预览
- **文件传输**：拖拽上传/下载，进度显示
- **文件操作**：复制、移动、删除、重命名
- **权限管理**：查看和修改文件权限
- **搜索过滤**：文件名搜索和内容搜索

**性能优化**：
- 懒加载大目录内容
- 文件图标缓存机制
- 异步文件操作
- 传输队列管理

### 5. 终端 (Terminal)

**定义**：终端是提供命令行界面的组件，每个会话拥有独立的终端实例。

**核心功能**：
- **Shell交互**：完整的SSH shell会话
- **命令执行**：实时命令输入和输出显示
- **历史记录**：命令历史浏览和搜索
- **多标签支持**：同一会话内的多终端标签

**状态管理**：
- **光标跟踪**：实时光标位置和样式
- **滚动缓冲**：可配置的输出历史行数
- **尺寸适配**：终端大小自适应容器
- **编码支持**：UTF-8等多字符集支持

**配置选项**：
- **外观设置**：字体、字号、颜色主题
- **行为设置**：复制粘贴、快捷键、滚动行为
- **性能设置**：缓冲区大小、刷新频率

### 6. AI助手 (AI Assistant)

**定义**：AI助手是智能辅助组件，为每个会话提供上下文相关的AI支持。

**功能模块**：

#### 6.1 对话式交互
- **自然语言界面**：支持中文和英文对话
- **上下文理解**：理解当前工作环境和任务
- **多轮对话**：保持对话上下文的连贯性

#### 6.2 命令辅助
- **命令建议**：基于历史和当前目录的智能建议
- **命令解释**：解释复杂命令的作用和参数
- **错误诊断**：分析命令执行错误并提供解决方案

#### 6.3 代码支持
- **代码分析**：分析代码文件并提供优化建议
- **语法检查**：检查代码语法错误
- **重构建议**：提供代码重构和改进建议

**上下文管理**：
- **工作目录感知**：了解当前文件系统位置
- **环境变量访问**：读取和使用环境变量
- **系统信息集成**：获取服务器系统信息
- **命令历史关联**：基于历史命令提供相关建议

**安全机制**：
- **命令白名单**：限制可执行的命令类型
- **敏感操作确认**：危险操作需要用户确认
- **权限检查**：验证操作权限和访问范围

## 架构关系图

```
应用层架构
├── 标签栏管理器 (Tab Manager)
│   ├── 标签页 1 → 会话 1 → 链接 A (生产服务器)
│   │   ├── 终端实例 1 (Shell Session)
│   │   ├── 文件管理器 1 (/var/www)
│   │   └── AI助手 1 (Web开发上下文)
│   ├── 标签页 2 → 会话 2 → 链接 B (测试服务器)
│   │   ├── 终端实例 2 (Database Shell)
│   │   ├── 文件管理器 2 (/home/test)
│   │   └── AI助手 2 (测试相关上下文)
│   └── 标签页 3 → 会话 3 → 链接 A (同一服务器，不同目录)
│       ├── 终端实例 3 (Git Session)
│       ├── 文件管理器 3 (/opt/app)
│       └── AI助手 3 (运维相关上下文)
```

## 数据流关系

```
用户操作 → 标签栏 → 会话管理 → 链接池 → 远程服务器
    ↓         ↓        ↓         ↓         ↓
UI更新 ← 组件状态 ← 会话状态 ← 连接状态 ← 服务器响应
```

## 技术栈映射

| 概念 | 技术实现 | 存储方式 |
|------|----------|----------|
| 链接 | SSH2 Client + 连接池 | SQLite + 内存缓存 |
| 会话 | 会话工厂 + 状态管理 | SQLite (持久化) |
| 标签栏 | Vue组件 + Pinia状态 | Electron Store |
| 文件目录 | SFTP + Vue组件 | 内存缓存 |
| 终端 | xterm.js + NodePTY | 内存缓存 |
| AI助手 | OpenAI API + Vue组件 | SQLite (对话历史) |

## 性能考虑

### 资源限制
- 最大并发链接数：15个
- 每终端缓冲区：1000行
- 单会话内存限制：50MB
- AI对话历史：100条消息

### 优化策略
- 懒加载非活动会话
- 连接池复用机制
- 内存使用监控和清理
- 缓存策略优化

这些定义为整个项目提供了清晰的架构指导，确保各个组件的职责明确、接口统一、性能可控。
