AI 直连：前端通过各厂商官方 SDK/REST 直连（支持流式/工具调用/JSON 模式）。

模型适配层（前端）：ModelProviderAdapter（OpenAI/AzureOpenAI/Anthropic/Google…）统一出：

createClient(config)、listModels()、chat(messages, options)、streamChat(...)

supports: { toolCalling: boolean, jsonMode: boolean, vision: boolean }

密钥与安全：浏览器无真正安全区；密钥仅存 localStorage（可选：用户口令本地 AES 加密），提供“易失模式（不落盘）”。

CORS/限流：使用厂商允许的前端直连域名；提供“每分钟请求上限 + 并发控制 + 指数退避”。

工具调用桥（前端编排器）：当大模型返回 tool call（结构化 JSON），前端把它路由到：

TerminalTool：向你的终端 WebSocket/后端 SSH 网关发命令

FileTool：向你的 SFTP/文件后端发操作

（可扩展更多工具）

注意：AI 直连不代表 SSH/SFTP 无后端。浏览器无法直接 SSH；这些仍由你现有的后端完成。AI 只是“发起请求”的策划与解释层。

高危命令防护：前端做“高危关键字 + 幂等识别 + 二次确认 + 干跑(dry-run)”流程。

可切换模型：UI 允许用户配置 provider、baseURL、apiKey、model、推理参数（温度、top_p、max_tokens）等。
